# Code Pattern Extractor

대규모 레거시 코드베이스에서 반복적인 코드 탐색 작업을 자동화하여  
코드 분석 시간을 수십 초에서 1초로 단축한 정적 분석 도구입니다.

---

## 🔒 코드 공개 및 구조 개선

본 프로젝트는 사내 레거시 시스템에서 사용된 코드를 기반으로 아이디어를 확장한 것입니다.

외부 공개를 위해 다음과 같은 작업을 수행했습니다:

- 실제 시스템 경로 및 식별 정보는 일반화하여 보안 이슈를 제거
- 기능 단위로 클래스를 분리하여 책임 기반 구조로 재설계
- 코드 분석 로직을 독립적인 모듈로 구성하여 재사용성과 확장성 개선

이를 통해 기존 스크립트 수준의 로직을 유지보수 가능한 구조로 개선했습니다.

## 🔍 문제

레거시 시스템 개발 과정에서 특정 기능(서비스 코드, 트랜잭션 ID 등)을 추적하기 위해
여러 화면 및 소스 파일을 반복적으로 확인해야 하는 비효율이 존재했습니다.

이 과정은 화면당 수십 초 이상이 소요되며,
대량의 화면을 처리할 경우 전체 개발 일정에 영향을 주는 병목 구간으로 작용했습니다.

---

## 🧩 해결 방법

이 문제를 해결하기 위해 정규식 기반 코드 패턴 분석 자동화 도구를 개발했습니다.

- 레거시 코드 구조를 분석하여 서비스 코드 위치 패턴 정의
- 정규식을 활용해 해당 패턴을 자동으로 추출
- ChatGPT를 활용해 패턴 정의 및 정규식 설계를 보조하고 로직을 개선

## ⚡ 성과
- 코드 분석 시간: 수십 초 → 1초로 단축
- 반복적인 코드 탐색 작업 제거
- 대량 화면 처리 시 개발 생산성 대폭 향상
- 개발자 작업 시간을 줄여 프로젝트 일정 안정성 확보에 기여

## ⚙️ 주요 기능

- 특정 메서드 호출 블록 추출
- 문자열 패턴 기반 코드 식별
- 마지막 파라미터 자동 추출
- 중복 제거 및 정렬 출력

---

## 🛠 기술 스택

- Java
- Regex (Pattern, Matcher)

## 📦 프로젝트 구조
```
extractor/
├── CodePatternExtractor.java
├── FileLoader.java
└── PatternConfig.java
```

각 클래스는 역할 기반으로 분리되어 유지보수성과 확장성을 고려한 구조로 설계되었습니다.


---

## ▶️ 실행 방법

```bash
javac -d out src/main/java/extractor/*.java
java -cp out extractor.Main
```
